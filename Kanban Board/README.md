A clean, interactive **Kanban Task Management Board** built using **Vanilla JavaScript**, designed to practice and demonstrate real-world **frontend development fundamentals** such as DOM manipulation, state management, and drag-and-drop interactions.

This project was created as a **skill-revision and hands-on learning exercise**, focusing on building functionality without relying on external libraries or frameworks.

2. Each task is rendered dynamically in the DOM.
3. Drag-and-drop events move tasks between columns.
4. Task data is synced to **LocalStorage** on every change.
5. On page reload, tasks are restored from saved state.
