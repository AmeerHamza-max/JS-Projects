# Snake Game

This is a simple and fully functional Snake Game built using **HTML, CSS, and JavaScript**.  
I built this project from scratch to practice real-world front-end development concepts like game logic, state handling, DOM manipulation, and data persistence.

## Features

- Classic snake movement using keyboard controls  
- Random food generation  
- Live score tracking  
- High score saved using localStorage  
- Timer that runs during gameplay  
- Restart game option  
- Clean and responsive grid-based layout  
- Separate game states (Start, Playing, Game Over)

---

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla)

## Installation

This project does not require any libraries or frameworks.

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/snake-game.git
````

2. Open the project folder.

3. Open `index.html` in your browser.

---

## How to Play

* Click **Start Game**
* Use arrow keys to control the snake:

  * Up Arrow → Move Up
  * Down Arrow → Move Down
  * Left Arrow → Move Left
  * Right Arrow → Move Right
* Eat the food to increase your score
* Avoid hitting the walls
* Try to beat your highest score, which is saved automatically
